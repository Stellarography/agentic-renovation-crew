Foreman
Coordinator/orchestrator for agent handoffs and final QA.
Recommended
• llama3.1:8b-instruct (fast)
• llama3.1:70b-instruct (max quality)
• Alt: mistral:instruct
Ollama pulls
ollama pull llama3.1:8b-instruct
ollama pull llama3.1:70b-instruct
ollama pull mistral:instruct
Tip: Use 8B for latency; promote to 70B for critical merges.
Coder
Code generation, refactoring, tests, and bug triage.
Recommended
• deepseek-coder:33b (strongest OSS)
• deepseek-coder:6.7b (lighter)
• qwen2.5-coder:32b or :7b
Ollama pulls
ollama pull deepseek-coder:33b
ollama pull deepseek-coder:6.7b
ollama pull qwen2.5-coder:32b
ollama pull qwen2.5-coder:7b
Use larger coder models for complex multi-file edits and reasoning.
Planner
Decomposes tasks, proposes options, drafts execution steps.
Recommended
• mistral:instruct (snappy)
• llama3.1:8b-instruct (balanced)
Ollama pulls
ollama pull mistral:instruct
ollama pull llama3.1:8b-instruct
Constrain outputs to bullet steps, owners, and rough estimates.
RAG
Retrieval, embeddings, and grounded answering.
Recommended
• Embeddings: mxbai-embed-large (1024D)
• Embeddings: nomic-embed-text (768D)
• Gen: llama3.1:8b-instruct
Ollama pulls
ollama pull mxbai-embed-large
ollama pull nomic-embed-text
ollama pull llama3.1:8b-instruct
Chunk ~800–1,000 tokens, overlap 100–200; store vectors in your DB/FAISS.
Vision
Image understanding, OCR-lite, charts, and UI parsing.
Recommended
• qwen2.5vl:7b (multimodal VLM)
• llava or llava-llama3
Ollama pulls
ollama pull qwen2.5vl:7b
ollama pull llava
ollama pull llava-llama3
Provide task-specific system prompts; prefer PNG/JPG ≤2MP for speed.
Speech
Speech-to-text (ASR) for meetings, commands, and notes.
Recommended
• ASR: Whisper family (local)
• TTS: external (Piper, Coqui XTTS)
Ollama pulls
# Example community build (ASR)
ollama pull dimavz/whisper-tiny
For higher accuracy, consider faster-whisper or Whisper large-v3 via non-Ollama runtimes.
Usage notes
  • Quantized variants (e.g., q4_0, q5_K_M) reduce VRAM at minor quality trade-offs.
  • Prefer instruct-tuned tags for agents and chat-style prompting.
  • For RAG: test both mxbai-embed-large and nomic-embed-text; results vary by corpus.
  • Pin versions in production: e.g., ollama pull llama3.1:8b-instruct-q4_0.
Context
Llama 3.1 supports up to 128K input in many tags.
Latency
Use 7B–8B for interactive loops; scale up for final pass.
Vision IO
Send image bytes/base64; keep resolutions modest for speed.
Emb dims
mxbai: 1024D · nomic: 768D
